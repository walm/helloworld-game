Hello World - The Game
=====================

![Screenshots](https://raw.github.com/walm/helloworld-game/master/screenshots.jpg)

Save the World from all Hello'ers, fire rockets at their space ships.

## Setup

    ./scripts/build.sh

*This will run*

    bundle install
    pod setup
    pod install
    xcodebuild ...
